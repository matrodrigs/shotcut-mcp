Capabilities

Real timeline editing.

Work directly with editable MLT project files while Shotcut supplies the render engine, codecs, filters, and effects.

Apply up to 500 related timeline operations as one validated transaction with revision checks, locks, backups, and atomic replacement.

02

Complete timelines

Add and arrange tracks, media, generators, gaps, transitions, effects, markers, notes, and subtitles through structured MCP tools.

Frame-accurate review

FAQ

Common questions.

What is Shotcut MCP?

Shotcut MCP is an independent, open-source Model Context Protocol server for AI-assisted video editing. It exposes structured tools that create and modify Shotcut-compatible MLT project files.

Does it control the Shotcut interface?

No. It edits saved project files directly and uses the local Shotcut, Melt, FFmpeg, and FFprobe executables for validation, previews, and rendering.

Does it upload my videos?

No. The MCP server runs locally over stdio. Media and project files remain on your computer unless your chosen AI client has separate upload behavior.

Is Shotcut MCP free?

Yes. The project is released under the MIT License. Shotcut and the MLT framework are separate open-source projects.
